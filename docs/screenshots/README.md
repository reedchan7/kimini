# README screenshots

Public screenshots must use synthetic workspaces, paths, sessions, messages,
models, and account state. Never capture a real Kimi Code session and redact it
afterward.

The 0.3.0 images in this directory were captured from the packaged native app
connected to an isolated loopback demo service. They contain no user session or
credential data.

- `kimini-native-overview.png` — primary product view
- `kimini-native-settings.png` — native settings surface
